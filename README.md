# dungeonHack

A simple Moria-like dungeon crawler written in C with SDL2.

## Building

### Linux

```sh
make
```

### Windows

```sh
make -f Makefile.win
```

## Controls

- Arrow keys: Move
- `r`: Wait/rest a turn
- Move onto `>`: Descend stairs
- `ESC`: Quit the game
